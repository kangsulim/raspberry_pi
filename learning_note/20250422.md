### Collection

1. Set(집합)

    순서 X, 중복 X

    중복제거 -> SQL(RDBS)

2. Array(배열)

    순서 O, 중복 O

3. List (linked : 단방향 / double linked : 양방향)

    순서 X, 중복 O

4. Map(Dict)

    순서 X, 중복 X

    key : value
```py
Dict = {
  "a" : "무언가",
  ...
}
```
